# Jump Diffusion Playground

Open index.html in a browser.

Features:
- Brownian motion (sigma slider)
- Poisson jump events (lambda slider)
- Jump size control
- Absorbing boundaries
- Particle visualization
- Transition density histogram
- Survival curve
- Spawn-at-x0 experiment

Next extensions:
- Numerical vs ML comparison
- Probability-flow ODE visualization
- Exact B1/B2 reproduction from the paper
